// Guía técnica · Cómo enviarlo

Configurar y
enviar cada edición.

Paso a paso con Brevo (gratuito). No necesitas saber de código — todo desde un panel visual.

PARTE 1 — CONFIGURACIÓN INICIAL (una sola vez)
1

Crear cuenta en Brevo

Ve a brevo.com y regístrate con tu correo de Aegis8. El plan gratuito incluye hasta 300 correos/día y contactos ilimitados — suficiente para empezar. No necesitas tarjeta de crédito.

2

Crear la lista "Docentes Aegis8"

En Brevo → Contactos → Listas → Nueva lista. Nómbrala "Docentes Aegis8". Anota el número de ID que aparece debajo del nombre — lo necesitarás en el paso 4.

3

Obtener la API Key

En Brevo → Configuración (ícono arriba derecha) → API Keys → Generar nueva clave API. Nómbrala "Aegis8 Web". Copia la clave — solo se muestra una vez. Guárdala en un lugar seguro.

4

Pegar las credenciales en este archivo

Abre newsletter.html en VS Code. Busca estas dos líneas al final del archivo y reemplaza los valores:

// Busca estas líneas y reemplaza:
const BREVO_API_KEY = 'TU_API_KEY_AQUI';
const LIST_ID = 3; // reemplaza 3 por el ID de tu lista
5

Verificar el dominio remitente (importante)

En Brevo → Configuración → Remitentes y dominios → Agregar dominio. Agrega aegis8.org y sigue las instrucciones para agregar 2 registros DNS en tu proveedor de hosting. Esto evita que tus correos lleguen a spam y es obligatorio para envíos masivos.

6

Crear el correo de bienvenida automático

En Brevo → Automatización → Nuevo workflow → "Contacto agregado a lista". Configúralo para que envíe automáticamente un correo de bienvenida cuando alguien se suscriba. Incluye el PDF de grooming como primer recurso — fideliza desde el día uno.

PARTE 2 — CREAR Y ENVIAR CADA EDICIÓN (mensual o semanal)
1

Definir la amenaza del mes

Revisa las fuentes: PDI Chile (pdichile.cl), CSIRT.gob.cl, y redes de docentes. Elige la amenaza más relevante para el nivel escolar. Para básica: grooming, ciberacoso. Para media: sextorsión, phishing, identidad digital.

2

Crear la campaña en Brevo

En Brevo → Campañas de email → Crear campaña → Newsletter. Asunto recomendado: Aegis8 [Mes] · [Nombre amenaza breve]
Ejemplo: Aegis8 Abril · Phishing disfrazado de JUNAEB

3

Usar la plantilla guardada

La primera vez diseña el correo siguiendo la estructura del preview de esta página: encabezado Aegis8, sección "La amenaza", sección "En el aula esta semana", sección "Para llevar a casa" y botón de descarga PDF. Guárdala como plantilla — las siguientes ediciones solo cambias el contenido, no el diseño.

4

Seleccionar la lista y programar el envío

Selecciona "Docentes Aegis8". Programa para el primer lunes del mes a las 08:00 — mejor tasa de apertura para docentes chilenos según datos de email marketing LATAM. Brevo te permite programar con anticipación.

5

¿Mensual o semanal?

Mensual recomendado para empezar — menos carga de contenido, menos desinscripciones, mayor calidad por edición. Si en 3 meses tienes más de 500 suscriptores y quieres aumentar la frecuencia, puedes hacer un correo semanal más breve: solo la amenaza activa + una pregunta para el aula. Guarda el plan completo con actividad para el envío mensual.

TIP · MÉTRICAS QUE IMPORTAN

En Brevo puedes ver tasa de apertura, clics y desuscripciones por edición. Un buen newsletter educativo tiene 35–45% de tasa de apertura. Si baja del 25%, el asunto del correo necesita más contexto urgente o el contenido no es lo suficientemente específico. El indicador más importante es cuántos hacen clic en "Descargar plan de clase" — ese es el valor real que entrega Aegis8.
